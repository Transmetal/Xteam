## INSTALLATION [Termux]

* `apt update`
* `apt upgrade`
* `pkg install python`
* `pkg install python2`
* `pkg install git`
* `git clone https://github.com/Transmetal/Xteam`
* `ls`
* `cd Xteam`
* `pip3 install -r requirements.txt`
* `chmod +x *`
* `bash setup.sh`
* `bash Xteam.sh`
